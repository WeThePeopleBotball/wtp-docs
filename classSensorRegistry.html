Detailed Description

Centralized manager for analog, digital, and calibrated sensors, including gyro support.

This registry allows you to register sensors with string identifiers, read their values, and store/retrieve calibration data. It also includes utility methods for working with the gyro.

Example:

SensorRegistry registry;
registry.create_analog("line_left", 0);
registry.create_digital("button", 1);
auto value = registry.get_analog_value("line_left");
auto pressed = registry.get_digital_value("button");
