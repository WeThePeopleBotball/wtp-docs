Detailed Description

Abstract base class for robot movement using different drive systems.

This interface defines commands for both omni-directional and differential drive systems. It supports relative and absolute turning, square-up routines, and vector-based control.

  • The left() and right() methods refer to lateral omni movement (e.g., on Mecanum).
  • The left_side() and right_side() methods control only the motors on one side and are typically used for squaring up against walls or controlled pivots.
