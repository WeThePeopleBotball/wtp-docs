Detailed Description

Aligns the robot with a wall using bumper sensors to recalibrate its orientation.

The WallSquareUp class aligns the robot by driving both sides forward until bumper sensors indicate contact with a wall. Once both bumpers are pressed, the robot is considered aligned.

Sensor Requirements

This class depends on two digital sensors registered in the SensorRegistry:

  • "squareup-left": left bumper
  • "squareup-right": right bumper

These sensors must return true when their respective button is pressed.

Usage Flow

  1. Call execute(speed) inside a loop or once per control tick.
  2. Monitor the return value to detect successful alignment.

This class is implemented as a singleton.
