Detailed Description

Servo control with position and degree interfaces, easing functions, and threaded smooth movement.

ImprovedServo wraps a physical servo and allows:

  • Moving to a tick position or angle in degrees
  • Applying easing functions for smooth movement (e.g., ease-in-out)
  • Executing movement in a background thread, so set_position and set_degrees are non-blocking
  • Using a calibrated degree-to-tick map for more accurate positioning

You can wait for the motion to finish using wait_until_stopped().

Example:

ImprovedServo servo(0);
servo.add_point(0.0f, 2047);
servo.add_point(180.0f, 0); // Supports reversed ranges
servo.set_degrees(90); // Moves using mapped ticks
servo.wait_until_stopped(); // Wait for completion
