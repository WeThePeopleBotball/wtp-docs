void terminate_after ( std::chrono::seconds timeout)

Starts a watchdog timer that terminates the program after the specified timeout.

This function starts a background thread that waits for the given amount of time. If the program is still running after the timeout, it will forcibly terminate (e.g., using std::exit or std::abort).

This is useful for:

  • Preventing infinite loops during testing
  • Failsafe shutdowns in autonomous systems
  • Enforcing hard real-time constraints
Parameters
timeoutDuration to wait before forcibly terminating the application.
Note
Once started, this watchdog cannot be stopped or reset. It will always trigger after the specified time.

Example:

#include "watchdog.hpp"
#include <iostream>
#include <thread>
int main() {
terminate_after(std::chrono::seconds(5)); // Will terminate after 5 seconds
while (true) {
std::cout << "Running..." << std::endl;
std::this_thread::sleep_for(std::chrono::seconds(1));
}
return 0;
}
