◆ until_light()

void until_light ( int port,
int max_waiting = 0 )

Waits until a bright light turns on to start the robot.

This function performs an interactive two-step calibration:

  1. Calibrate the light ON value by placing the light on the sensor and pressing the side button.
  2. Calibrate the light OFF value by covering the sensor and pressing the side button again.

Then it waits until the sensor detects a significant increase in brightness.

Parameters
portAnalog sensor port to use (e.g., 0 for analog(0)).
max_waitingTimeout in milliseconds. If 0, no timeout is applied.
