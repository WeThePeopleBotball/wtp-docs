Detailed Description

Base class managing a predefined program lifecycle with overridable stages.

The lifecycle consists of the following stages (in order):

  • Calibration
  • Waiting
  • Execution (non-overridable)
  • Cleaning
  • Resetting

Derived classes can override each stage except the execution, which is defined by tasks.

void Lifecycle::run ( )

Executes the entire lifecycle in sequence.

Executes the full lifecycle in the correct order: calibrate → wait → execute_tasks → clean → reset.

The order of execution is:

  1. calibrate()
  2. wait()
  3. execute_tasks()
  4. clean()
  5. reset()
