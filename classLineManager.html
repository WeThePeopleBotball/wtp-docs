Detailed Description

Manages two light sensors for line-following, crossing detection, and proportional correction.

The LineManager handles black line following using two light sensors (left and right). These sensors must be registered in the SensorRegistry as analog sensors under the keys:

  • "lf-left" for the left sensor
  • "lf-right" for the right sensor

This class uses a MotorController to steer based on a proportional control algorithm.

Before executing line_follower(Side), you must call calibrate(Side) to compute the correction value for the chosen sensor side. This calibration is not for the sensors themselves, but for preparing internal tuning data used during runtime.

This class is implemented as a singleton.

◆ calibrate()

void LineManager::calibrate ( Side s)

Prepares the line follower correction for a given sensor side.

This method must be called before using line_follower(Side) or line_follower_backward(Side) to initialize the internal correction logic for the chosen side.

Note
This method does not calibrate the sensors themselves. Sensor calibration (e.g., storing white/black levels) must be done separately in SensorRegistry.
Parameters
sSide (left or right) that will be used during line following.
