Detailed Description

Stores and retrieves median-calibrated values for sensors.

This class takes a set of sensor sampling functions and performs multiple readings for each. The median of these readings is stored as the calibration value for that sensor.

Example usage:

CalibrationStore store({
{"front_ir", [] { return analog(0); }},
{"line_sensor", [] { return analog(1); }}
});
unsigned short calibrated = store.get_value("front_ir");
