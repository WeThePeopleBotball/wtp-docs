void sender ( )

Sends a UDP heartbeat message with the robot's current navigation state.

This function constructs a JSON-formatted message containing the robot's current node, the path it is following, and an offset vector indicating its fine position adjustment. The message is sent over UDP to a predefined target (e.g., a broadcast address).

Example JSON structure:

{
"current_node": 12,
"path": [12, 13, 14, 15],
"offset_vector": { "x": 0.05, "y": -0.02 }
}
Note
This function is typically called at regular intervals (e.g., every 100ms).
